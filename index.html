Thoughts on Progressive Enhancement

Progressive enhancement is the practice of priortizing content instead of technology. Instead of focusing on more advanced browsers like with Graceful degradation. With Graceful degradation, the best browsers get the newest and best versions while others get the passable but not so great versions of the content Progressive enhancement focuses on content by splitting up the three aspects of a web page and if a device so chooses to accsess one or more of those aspects it can do so. Sometimes a device wants to only look at the HTML like search engines or screen readers. a lot of times a device may want to see the CSS so the user can see the designs. other times a user may want to have some functionality, which can be done with Java script. the point is instead of versions for the best browsers, we should allow each browser to accsess what they want or need equally. if a device can use javascript then it can use it, otherwise HTML and CSS should work just fine for that device. likewise when a user depends on the developers HTML skills, they deserve to be prioritized too.
